# Worm

Finds daily popular tweets.

#### worm.properties

```properties
statusLimitToKeep=30
minFollowingCount=50
maxFollowingCount=2000
maxFollowersCount=200000
languageKey=tr
quoteLimit=13
quoteHour=22
quoteMinute=00
```
